# Rainbow Tour

## Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/RAshi230227/Rainbow-tour.git


2. Open XAMPP.

3. Start Apache and MySQL.

4. Go to http://localhost/phpmyadmin/index.php in your browser.

5. In phpMyAdmin, go to the SQL tab. From the project folder, open Rainbow-tour/db/ddl.php and Rainbow-tour/db/dml.php, then copy and paste their contents into the SQL query window in phpMyAdmin and click Go.

6. Finally, go to http://localhost/rainbow-tour/tourist/ in your browser.
